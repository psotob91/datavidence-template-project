# Module: evidence synthesis (review / meta-analysis)

Enabled because `synthesis` is in `modules`. For systematic reviews / meta-analyses.

- **Frameworks:** report with **PRISMA**; appraise review with **AMSTAR-2 / ROBIS**;
  study-level risk of bias (e.g., RoB2) visualized with `robvis`.
- **Method:** PRISMA flow (**`PRISMA2020`**); pooled estimates + heterogeneity; forest
  and funnel plots; pre-registered protocol.
- **Packages:** `meta`, `metafor`, `robvis`, `PRISMA2020`.
