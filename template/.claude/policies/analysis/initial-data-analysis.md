# Initial data analysis (IDA)

Check the data is fit for purpose before modeling — without peeking at the
exposure–outcome association.

## Rules

- **Run a structured IDA** (STRATOS framework) before modeling: data integrity,
  structure, missingness, distributions, and the assumptions the planned analysis
  needs.
- **IDA ≠ EDA.** IDA checks fitness-for-purpose; it does not hunt for findings.
- **Record it.** IDA results justify the cleaning and modeling choices that
  follow.
