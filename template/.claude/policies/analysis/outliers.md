# Outliers

Extreme values are inspected and decided on, never silently removed.

## Rules

- **Detect → inspect → handle, and document each step.** Never delete a point
  without a recorded reason.
- **Distinguish error from signal.** A data-entry error may be fixed or removed
  (with justification); a genuine extreme value is usually kept.
- **Sensitivity when it matters.** If handling changes a result, run the analysis
  **with and without** and report both.
